## 🌟 项目简介

这是一个用于处理图片文件的 Node.js 脚本。脚本会读取当前目录下名为 `images` 的文件夹，筛选并输出所有大小超过 **2MB** 的图片文件名。

## 🛠️ 运行环境

- **Node.js**：确保已安装 Node.js（版本 10 以上）。
- **文件结构**：项目根目录下需包含一个 `images` 文件夹。

## 📂 文件结构

```bash
├── listLargeImages.js   # 核心脚本文件
└── images/              # 放置待筛选的图片
    ├── img1.jpg
    ├── img2.png
    └── ...
```



## 🚀 使用方法

1. **克隆/下载项目**：将项目文件下载到本地。
2. **打开终端**：进入项目根目录。
3. **运行脚本**：
   ```bash
   node listLargeImages.js
   ```


## 📋 输出示例

- **若存在符合条件的图片**：
  ```
  以下图片大小超过 2MB:
  - large_image1.jpg
  - big_picture2.png
  ```


- **若不存在符合条件的图片**：
  ```
  未发现大小超过 2MB 的图片。
  ```


---

### 📌 注意事项

- **非递归搜索**：脚本只搜索 `images` 文件夹第一层目录，不会进入子文件夹。
- **支持格式**：默认支持 `.png`, `.jpg`, `.jpeg`, `.gif`, `.bmp`, `.webp`, `.svg` 等常见图片格式。
- **阈值设置**：如果您需要更改文件大小阈值（例如筛选 1MB 或 5MB 以上的图片），请编辑脚本中的 `SIZE_THRESHOLD` 常量：
  ```javascript
  const SIZE_THRESHOLD = 2 * 1024 * 1024; // 2MB
  ```
